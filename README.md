\# Analysis of Microalgal Communities



\## Overview

This repository contains R scripts and algal genus pollution index (Palmer index values) data for ecological diversity analysis and creating interactive graphs. 📊📈 You can use this code for graph preparation, but first, you need to calculate the mean and standard deviation of your data. If you want new R code that also helps you calculate the mean and standard deviation of your data, you need to add that section and then redraw the graphs. 💻📊



\## Methods

\- Diversity indices

\-Similarity triangular heatmap (Jaccard similarity index) with NDMS calculation

\- CCA analysis

\- Graph generation



\## Structure

\- scripts/: R codes

\- data/: raw datasets

\- figures/: generated plots
